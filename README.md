## 🚀 Projects

### 🔍 Oculus Auditor  
Cybersecurity-focused web application auditing tool that evaluates websites for misconfigurations, exposure risks, and compliance gaps.  
Analyzes HTTP headers, cookies, server disclosures, and exposed secrets while generating audit-ready reports and real-time dashboards.

---

### 🛡️ Sentinel Defense Platform  
Hybrid NIDS + SOC simulation platform combining signature-based detection with ML-based threat analysis (XGBoost).  
Detects SQL injection, XSS, and command injection using a multi-stage detection pipeline and intelligent threat scoring.

---

### 🐝 Credentia – Honeypot-Based Firewall  
Lightweight Python-based firewall integrated with a honeypot system to detect, monitor, and deceive attackers.  
Includes process monitoring, suspicious file detection, virus scanning, and attacker behavior logging.

---

### 🎯 Insight – Phishing Detection System  
Detects phishing and obfuscated URLs, including trailing and disguised links.  
Focuses on URL analysis and real-time threat detection.

Includes hands-on labs, attack simulations, and real-world case studies.
